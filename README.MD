## Instrucciones de la evaluación

Para revisar las instrucciones completas de la evaluación y su rúbrica, haga clic en el siguiente enlace:

[https://pablot18.github.io/icc-est-examen-final-suple-b/](https://pablot18.github.io/icc-est-examen-final-suple-b/)

---

### 1. Ingreso al entorno de trabajo

1. Abrir el VS Code que se ejecuta dentro del contenedor Docker.
2. En el navegador, acceder a:

```
http://localhost:8080
```

o al puerto configurado al momento de iniciar el contenedor.

---

### 2. Preparación del repositorio

1. En una nueva pestaña del navegador, abrir GitHub.
2. Realizar un **fork** del repositorio del docente:

[https://github.com/PabloT18/icc-est-examen-final-suple-b](https://github.com/PabloT18/icc-est-examen-final-suple-b)

3. En la pestaña del VS Code del contenedor, clonar **su propio repositorio** (el fork), no el repositorio original del docente.

---

### 3. Inicio del examen

Una vez clonado el repositorio, realizar un primer commit con el siguiente mensaje:

```
inicio examen
```

Luego, revisar nuevamente las instrucciones publicadas en:

[https://pablot18.github.io/icc-est-examen-final-suple-b/](https://pablot18.github.io/icc-est-examen-final-suple-b/)

Proceder con el desarrollo del examen.

---

### 4. Finalización del examen

Al concluir el desarrollo, realizar un commit con el siguiente mensaje:

```
fin examen
```

Si necesita realizar correcciones adicionales después del primer commit de finalización, debe registrar nuevos commits con los siguientes mensajes:

```
fin examen 2
fin examen 3
...
```

según corresponda.
